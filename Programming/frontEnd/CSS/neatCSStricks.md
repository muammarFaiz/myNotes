1. calling
    - .myclass[type="checkbox"]

        it is calling the dom element that have a class .myclass and type = checkbox.
    - -webkit-linear-gradient

        ```css
        body {
          background-image: -webkit-linear-gradient(65deg, #A683E2 50%, #E7E9FD 50%)
        }
        ```
        making a background color with linear gradient, with 65 degree angle, with only two color / 50% - 50%.
    - .item:last-child {}

      selecting the last child of the element that have class .item.
    - div + p

      select all the `<p>` elements that are placed immediately after `<div>` element.
    - ::placeholder

      "pseudo selector" placeholder select the placeholder text so you can edit it.
